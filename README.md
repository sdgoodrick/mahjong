A work-in-progress but fully playable implementation of mahjong solitaire in
C++23 and SDL3!

# Features

- full undo and redo!
- visible available match counts along with notice of win/lose state
- cycle through hints showing all available matches
- parse different layouts/board configurations
- configurable settings (hard-coded, see in-progress)

# In-progress

- UI for real-time configuration
- Save/Load

# Build instructions

```sh
mkdir build && cd build
cmake ../
make
```
